Introduction

FusionFall Fighters is a fan game that explores side stories in the universe featuring original concepts as well as some concepts planned for the original run of the game as well as the now cancelled fan revivals FusionFall Retro and FusionFall Legacy. It will be an RPG made in RPG Maker MV. It was started by RadioRave, and continued by Zakthepower.

It will be an episodic game with each episode having a playable character that was not previously playable. Each episode will have its own story, but we plan to connect them all in some way.



Gameplay

It will play much like the original FusionFall, but with some key differences:

(One) - There will be a larger focus on story

(Two) - You will play as a certain character depending on the episode you play

(Three) - The combat will be turn-based

(Four) - It will be a standalone singleplayer experience.


Story

Episode One: Boomer And The Blowfish


Boomer, Bubbles, and Jackie Khones have discovered Fuse`s plans for the townsville zoo, and the blowfish. They have to find the blowfish and stop fuse`s plan before it is too late...


Release Date

The game has no set release date, since it is episodic. We can inform, however, that episode one is currently in beta, and playable! If you would like to participate in beta, please join our discord server and follow the directions!
