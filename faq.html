FAQ (Frequently Asked Questions)


Q: What is this game?


A: See the About page



Q: Who made it?


A: You can find our full team in the About page



Q: What engine was the game made in?


A: We used RPG Maker MV



Q: When will the game come out?


A: The game has no set release date, since it is episodic. The demo for Episode One is out! The full release and other episode release dates are to be confirmed.



Q: What will you do if you get a DMCA?


A: We got that covered, no need to worry



Q: Can you play this game online?


A: No, it is a singleplayer RPG



Q: Who did the (great) art?


A: Piyush is our great artist ;)



Q: Where can I find the latest updates on the game?


A: The homepage will display the latest showcase video. You can also check the devblogs page or the #announncements channel in the discord



Q: Is this a FusionFall revival project?


A: In a way, yes. However, we do not plan to make a 3D game or even remake the game. This game is just set in the same universe, but it has a lot of differences



Q: Can I use the game assets in my own game?


A: No
